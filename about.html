About This Website

Made for Primary School

This website was created to support Primary School students learning about machine learning and artificial intelligence through Google's Teachable Machine.

The unit uses simple, scaffolded instructions with plenty of visual support — designed so every student can succeed. No prior programming knowledge is needed.

Unit Summary

Over 6 lessons, students will:

  • Build image classification models using hand-drawn shapes
  • Explore bias and fairness in AI
  • Learn how to properly test and evaluate a model
  • Design their own AI product idea
  • Reflect on the ethics and limitations of AI

No camera, microphone, or coding is required. Students use Microsoft Paint to create images and Google Teachable Machine (in-browser) to train and test models.

Technical Details

  • Offline-friendly: This site works by opening index.html directly in a browser — no server needed.
  • No external dependencies: All CSS and JavaScript is included locally. No CDNs, no frameworks.
  • Data storage: Student answers are saved using the browser's localStorage. Data stays on the student's computer and is not sent anywhere.
  • Print-friendly: Each lesson page has CSS print styles for creating worksheets.
  • Accessible: High contrast, large fonts, keyboard navigation, skip links, and semantic HTML.
